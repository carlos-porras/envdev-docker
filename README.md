# Develop Enviroment for Docker

## 1. Steps

Clone Repository

```bash
 cd ~ 
 git clone <repo-url> -b dev
```

Create missing folders

```bash
 mkdir -p ~/www/php
 mkdir -p ~/mysql/data
```

Go to the folder project

```bash
 cd ~/envdev-docker
```

Run develop environment

```bash
 docker-compose up -d
```
